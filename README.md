# FunBets for TizenBrew

FunBets Arcade launcher module for TizenBrew on Samsung TVs.

## TizenBrew Module

If installed from TizenBrew's GitHub module screen, add this exact value:

`kefrulz/funbets-tizenbrew@1.0.1`

Do not type the `gh/` prefix yourself on the GitHub module screen. TizenBrew adds that internally.

If installed by URL, add:

`https://cas.oqyb-drtbrt-aux2.online/tizenbrew/funbets/`

## Launch Target

This launches:

`https://cas.oqyb-drtbrt-aux2.online/?tv=1&app=tizenbrew`

## Files

- `package.json`: TizenBrew app module manifest.
- `app/index.html`: Small fullscreen launcher that redirects to FunBets TV mode.
